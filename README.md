# SMS Reseller Platform

A full-stack SMS reseller web application that allows users to rent virtual phone numbers, receive OTP messages in real time, and manage rentals through a secure dashboard.

This platform is designed for resellers, startups, and businesses that require temporary or single-use SMS numbers across multiple countries.

---

## Key Features

- User authentication (login & registration)
- User and Admin dashboards
- Rent virtual phone numbers
- Real-time OTP receipt handling
- Single-use and reusable number support
- Message history and status tracking
- Admin-controlled number management
- Mobile-responsive UI

---

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)

### Other Tools
- Git & GitHub
- RESTful API architecture

---

## System Architecture

- Frontend communicates with backend via REST APIs
- Backend handles authentication, number rentals, OTP processing
- MongoDB stores users, numbers, rentals, and messages
- Admin panel manages number availability and pricing
